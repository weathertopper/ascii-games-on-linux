# Zork

## Setup

Run the following:

```bash
./setup_zork.sh
```

### Play

Run the given bash script to play Zork I, Zork II, or Zork III:

```bash
./start_zork.sh i
```

... where `i` is `1`, `2`, or `3`, depending on the desired game.
